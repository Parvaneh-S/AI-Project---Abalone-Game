## How It Works

1. User launches the game
2. User clicks through the landing page to the board layout page
3. User clicks the "German Daisy" button (middle area)
4. User selects their color (black or white circle at the bottom)
5. User clicks "Next"
6. The board is rendered with the German Daisy configuration

## Color Inversion Support

The implementation automatically handles color inversion:
- If user selects WHITE circle: White marbles in their designated positions
- If user selects BLACK circle: Colors are inverted

This is handled by the `invert_colors` parameter in the BoardRenderer.
